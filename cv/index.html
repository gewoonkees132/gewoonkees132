Summary

I solve the problems that stand between a design and a built structure. Building engineer by training, I spent six years at Vertico carrying robotic fabrication projects from first conversation to delivery, learning each discipline as the work demanded. Every project ran into the same underlying problem: translation between architect and engineer, between design and machine, between what is possible and what is not yet possible. Closing those gaps is how the resource-efficient buildings we can only render today actually get built.
